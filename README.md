# 📚 Lumina Archives — Library Management System

A modern, responsive **Library Management System** built using **HTML, CSS, and JavaScript**. The application provides an intuitive interface for managing books, library members, borrowing records, and administrative tasks while demonstrating clean UI design and client-side data management.

---

## ✨ Features

### 🔐 Authentication
- User Registration
- Secure Login System
- Password hashing using SHA-256
- Persistent user sessions

### 👥 Role-Based Access
- Admin Dashboard
- Patron Dashboard
- Role-specific navigation and permissions

### 📖 Book Management
- Add new books
- Edit book details
- Delete books
- Restock available copies
- Search books
- Filter by category
- Sort books

### 📚 Borrowing System
- Borrow books
- Return books
- Automatic due date calculation
- Fine calculation for overdue books
- Hold/Reservation system
- Borrowing history

### 📊 Dashboard
- Total Books
- Borrowed Books
- Overdue Books
- Fine Collection
- Category-wise collection visualization

### 🔔 Notifications
- Hold notifications
- User alerts
- Dismiss notifications

### 🎨 User Experience
- Responsive design
- Light/Dark theme toggle
- Accessible navigation
- Modern card-based interface
- Elegant library-inspired UI

---
